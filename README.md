# Gestión de Usuarios Renfe

Este repositorio contiene una práctica en **C** para gestionar usuarios en un sistema tipo **Renfe** (gestión de usuarios de un servicio). El proyecto forma parte de los ejercicios realizados en clase para aplicar estructuras de datos, manejo de ficheros y programación en C.

## Descripción

El código implementa un sistema básico de gestión de usuarios que permite:

- Dar de alta nuevos usuarios
- Eliminar usuarios existentes
- Listar todos los usuarios
- Guardar y recuperar datos desde archivos

El objetivo principal es practicar la manipulación de estructuras, ficheros y lógica de programación en C.

## Tecnologías utilizadas

- Lenguaje **C**
- Compilador compatible con C (por ejemplo, `gcc`)
